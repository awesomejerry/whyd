# Proposal: Daily Summary

## Problem
用戶每天記錄多筆成就，但缺乏一個集中的回顧頁面來總結今日的成就。晚上時段用戶需要一個方式來回顧當天的努力，並標記最重要的事情。

## Solution
建立一個每日回顧頁面（Daily Summary），提供：
- 今日成就總數統計
- 標籤分布視覺化
- 時間分布（最早/最晚記錄）
- 用戶可標記「最重要的事」
- 晚上自動提示功能（使用 localStorage 記錄）

## User Stories
1. 作為用戶，我想要在晚上看到今日成就總結，回顧一天的進展
2. 作為用戶，我想要標記今日最重要的一件事，強化成就感
3. 作為用戶，我想要看到標籤分布，了解今天的時間分配
4. 作為用戶，我想要關閉回顧後返回主頁面繼續記錄

## Impact
- 提升用戶的成就感和動力
- 幫助用戶反思每天的進展
- 增強產品的使用黏著度

## Scope
- 輸出檔案：`js/daily-summary.js`, `styles/main.css` (新增樣式)
- 觸發方式：點擊底部 🌙 按鈕
